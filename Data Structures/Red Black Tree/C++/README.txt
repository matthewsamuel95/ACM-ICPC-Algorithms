To run this code place all these file in the same folder.

Node.hpp -> It contains Node class, a node in a tree is the most fundamental unit and the tree is built from such nodes. Nodes are 
            the one that contains the actual data or refernce to data stored somewhere else.

Node.cpp -> Implementation of Node class.

RB_Tree.hpp -> It contains RBT class which is a self balancing binary tree. This is used by the user to implement a RB tree.

RB_Tree.cpp ->Implementation of RBT class.

menu.hpp -> This file consists of menu containing a user friendly descriptive way to interact with the program.

main.cpp -> This file contains the driver engine i.e., the main function.
